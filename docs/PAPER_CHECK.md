# Manuscript Check

The uploaded v2 manuscript is structurally coherent and technically disciplined. It clearly separates:

1. source-side structural memory from gravitational-wave memory,
2. scalar toy validation from full 3+1 GRHD implementation architecture,
3. synthetic detector/network separability from real detector sensitivity forecasts.

Main risks retained intentionally:

- The waveform proxy is diagnostic rather than a full relativistic waveform extraction.
- Nonzero memory templates are highly degenerate; the result supports memory-on versus memory-off separation, not precision inference of lambda_mem.
- The GRHD section is an implementation map, not a completed solver.

The only visible style issue in the current PDF is that most figures float to the end. This is acceptable for a preprint draft, but for journal submission it may be cleaner to move some figures nearer their first call or place figures after references as an explicit figure section.
